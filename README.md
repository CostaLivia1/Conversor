# Projet Convertisseur de Monnaie

## Description
Ce projet est une application simple de conversion de devises utilisant **HTML, CSS et JavaScript**. 
Vous pouvez convertir entre Euro, Real Brésilien, Dollar Américain et Livre Sterling.

## Fonctionnalités
- Sélectionnez la devise à convertir et celle cible.
- Entrez un montant et obtenez la conversion instantanée.
- Interface simple et intuitive avec logos des devises.
- Effets visuels au survol des images des monnaies.

## Technologies
- HTML5
- CSS3
- JavaScript
